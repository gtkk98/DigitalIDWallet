# Digital ID Wallet

A mobile-style digital wallet application built using Python and Kivy that allows users to securely store and manage personal identification documents such as national ID cards and driving licenses.

## Features

- Store multiple identification documents
- Add ID details including name, ID number, and image
- Scrollable card-style display of saved IDs
- SQLite database for persistent storage
- Secure PIN lock screen
- Multi-screen navigation using Kivy ScreenManager

## Technologies Used

- Python
- Kivy (Mobile UI Framework)
- SQLite
- PyCharm
